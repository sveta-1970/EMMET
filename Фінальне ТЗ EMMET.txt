Технічна задача: Реалізація HTML Emmet подібної функціональності на JavaScript
Опис задачі:
Вам потрібно розробити JavaScript програму, яка буде відповідати за конвертацію спрощеного синтаксису, подібного до Emmet, у фактичний HTML код. Програма повинна взяти введення користувача з одного вікна, а потім генерувати HTML відображення в другому вікні, відповідно до заданого синтаксису.


Аналог  https://pughtml.com/




Вимоги до функціональності:
Користувач вводить вхідний синтаксис у форматі .class>element#id, де:
.class представляє клас елемента.
> рівень вкладеності
^  на рівень вище
element представляє назву HTML елемента.
#id представляє ідентифікатор елемента.


Вхідний синтаксис може бути більш складним, але в даному завданні розглядається лише простий синтаксис до 3х рівнів вкладеносіт
Програма повинна перетворювати введений синтаксис на HTML репрезентацію та відображати її у другому вікні.
Вам потрібно коректно розпізнавати класи, ідентифікатори та інші атрибути, якщо вони задані в синтаксисі.


Програма повинна бути написана у вигляді функції або об'єкту, який має методи для обробки та відображення вхідного синтаксису.


Візуальний інтерфейс (вікна для введення та відображення результату) може бути зреалізований з використанням сторонніх бібліотек або власних засобів.


Приклад роботи програми:
Користувач вводить вхідний синтаксис: .one>.two
Програма перетворює синтаксис на HTML: <div class="one"><div class="two"></div></div>
HTML результат відображається у другому вікні.


Додаткові вимоги:
Програма повинна бути написана на чистому JavaScript без використання фреймворків, але ви можете використовувати стандартні API, такі як querySelector та createElement.
Код повинен бути добре структурованим та документованим.
Забезпечте обробку помилок, якщо введений синтаксис має неправильний формат або відсутній елемент для заданого класу / ідентифікатора.
Забезпечте можливість динамічного оновлення відображення результату при зміні введеного синтаксису.
Оптимізуйте програму для кращої продуктивності та взаємодії з користувачем.


Завершення:
Після завершення вам потрібно додати можливість введення синтаксису, його обробку та відображення результату. Впевніться, що ваша програма коректно перетворює спрощений синтаксис на HTML структуру з урахуванням класів, ідентифікаторів та атрибутів.
